Tipos mais comns de testes 
    - Testes de Unidade/ UnitTest
        > Testes Unitários Errado 
        > Unidade: menor parte testável de um software
        > Orientação a Objetos: classes
    - Testes de Integração
        Encontrar de integração entre as unidades, e não mais em 
        testar as funcionalidades da mesma 

        Integração entre unidades ou entre sistemas 
    - Testes Automatizados 
        "Simula" ações do usuário
        Aceitação: caixa preta 
        Regressão: garante integridade de versões passadas 
        Processo de automatização é caro: selecionar 
        funcionalidades 
Projeto Vaquinha 
    Projeto demostração de testes 
    ASP.Net Core MVC

> MVC: Model View Controller 
    - Model: Armazena, manipula e gera os dados 
    - View: Apresenta o conteúdo por meio da interface do
    usuário 
    - Controller: Cuida da interação do usuário, trabalham com o 
    model e, em última análise, selecionam uma view a ser 
    renderizada.
